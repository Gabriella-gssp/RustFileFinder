# RustFileFinder – Esempio (Linguaggi di Programmazione)

Questo progetto implementa una CLI in Rust per cercare file in una directory:

- ricerca per nome file (case-insensitive);
- ricerca per contenuto usando espressioni regolari (regex) su file di testo UTF-8;
- (opzionale) estrazione testo da PDF e ricerca nel contenuto;
- output in JSON o Markdown;
- statistiche di esecuzione (file analizzati, file ignorati, errori, ecc.).

Esempi di parole chiave: compilatore, interprete, semantica, tipi, Rust, Python.
